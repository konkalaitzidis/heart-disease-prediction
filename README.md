# Heart Disease Prediction

**Authors:** Evanthia Chatzigeorgiou, Konstantinos Kalaitzidis  
**Description:** ML pipeline to predict heart disease using clinical features. Implements Logistic Regression and Random Forest models with feature importance analysis.  

## Dataset

- Sourced from [Kaggle: Cleveland Heart Disease Data](https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data)  
- 918 patients, 14 features + target (binary: disease/no disease)
